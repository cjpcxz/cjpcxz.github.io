一个函数和它所捕获的变量\常量环境组合起来，称为闭包

  • 一般指定义在函数内部的函数
  • 一般它捕获的是外层西数的局部变量\常量

闭包本质查看

不捕获变量

  • 源码
  • 汇编
    • rax一般最为返回值，很明显将0xe5(%rip)的地址存放在计算器rax中，返回
    • 查看rax地址,很明显存放的是函数地址，显然不捕获变量的闭包，直接返回了函数的地址。并且和fu的指针一致。
    • 此外将xorl %ecx, %ecx进行异或,其值为0，并将结果赋值给edx（edx与rdx共寄存器，只是一个4字节，一个8字节),也可理解为赋值给了rdx
    • fn指针变量是16字节空间，
      • 其中赋值0x100002299 + 0xe337 = 0x1000105D0,0x1000022a0 + 0xe338 = 0x1000105D8,并且为movq(占8字节)移动
      • 因此，很显然这里将rax中的函数地址地址赋予fu，同时rdx也赋值给了fu，rdx和后续捕获变量有关系

        捕获变量

  • 源码
  • 汇编

    • 可以看出调用了swift_allocObject，在堆空间生成了对象。并且该对象的地址返回在rax中,打印出地址，以及存储的num,num2
    • leaq 0xf8(%rip), %rax，movq $0x9, 0x10(%rdx)、movq $0x3, 0x18(%rdx),函数地址传给了rax,生成的包装num 的对象传给了rdx,rdx和rax一般用做函数返回值。实际返回结果后，将rax和rdx上的值，赋值给了fn对象（由于在全局区执行，所以是rip，全局变量）
    • 注意，和未捕获的闭包表达式不同，捕获了变量的闭包，其赋值给rdx的函数地址，和实际的不同.(xcode旁边的注解和捕获变量的赋值，是不一样的)，实际上，该函数地址进行了一次包装

      捕获的时机

总结

  • 可以把闭包想象成是一个类的实例对象
    • 内存在堆空间
    • 捕获的局部变量\常量就是对象的成员（存储属性）
    • 组成闭包的西数就是类内部定义的方法

自动闭包

自动闭包的参数会延迟执行，甚至不执行

  • @autoclosure 会自动将20 封装成闭包{ 20 }
  • @autoclosure 只支持()一＞T格式的参数
  • 空合并运算符??使用了@autoclosure 技术
    • func ?? <T>(optional:T?,defaultValue:@autoclosure() throws -> T?) rethrows -> T?
  • 有@autoclosure、无@autoclosure，构成了函数重载
