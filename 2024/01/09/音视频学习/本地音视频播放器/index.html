本地音视频播放器

Created2024-01-09|Updated2024-01-18|音视频学习
|Post View:

总览

  1. 初始化SDL子系统
  2. 开始播放，开启子线程A读取文件信息
    1. 传入文件地址filename,创建解封装上下文_fmtCtx
    2. 从解封装上下文_fmtCtx中检索流信息
    3. 初始化音频信息
      1. 初始化解码器
        1. 根据解封装上下文找到音频流信息_aStream
        2. 根据音频流信息中的_aStream->codecpar->codec_id,找到合适的解码器AVCodec
        3. 根据解码器，初始化解码上下文
        4. 将音频流信息中的codecpar参数，复制到解码上下文
        5. 打开解码器
      2. 初始化音频重采样上下文
        1. 设定重采样输入/输出参数
        2. 根据输入/输出参数创建重采样上下文
        3. 初始化重采样上下文
        4. 创建重采样输入帧_aSwrInFrame和输出_aSwrOutFrame
        5. 初始化输出帧的_aSwrOutFrame的数据空间（输入帧_aSwrInFrame会在解码的时候，自动获取合适的数据空间）
      3. 初始化SDL
        1. 初始化SDL音频参数,即重采样输出的参数
        2. 打开音频设备
    4. 初始化视频信息
      1. 初始化解码器
        1. 根据解封装上下文找到视频流信息_vStream
        2. 根据视频流信息中的_vStream->codecpar->codec_id,找到合适的解码器AVCodec
        3. 根据解码器，初始化解码上下文
        4. 将视频流信息中的codecpar参数，复制到解码上下文
        5. 打开解码器
      2. 初始化像素格式转换上下文
        1. 设定格式转换输入/输出参数
        2. 根据输入/输出参数创建重采样上下文
        3. 创建格式转换输入帧_vSwsInFrame和输出帧_vSwsOutFrame
        4. 初始化输出帧的_vSwsOutFrame的数据空间（输入帧_vSwsInFrame会在解码的时候赋值)
    5. 启动SDL音频子线程，开始播放声音
    6. 启动视频子线程，开始解码视频
    7. 从解封装上下文中，读取音视频数据
    8. 资源释放标记
  3. 资源的释放
  4. 视频画面的显示
